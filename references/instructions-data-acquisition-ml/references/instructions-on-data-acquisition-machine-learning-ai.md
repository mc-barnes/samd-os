Instructions on Data Acquisition
The Instructions on Data Acquisition define what kind of data is collected and
annotated (prepared) for machine learning model development. The require-
ments for general data acquisition can be defined less specific than the subset
of data used for annotations as any additional data can be used for future de-
velopments that are not specified yet.
