# Engineering

Bug investigations, RFCs, and IEC 62304 software lifecycle documentation.

## Doc index
- bugs/    — Bug investigation reports and root cause analyses.
- rfcs/    — Request for Comments — technical design proposals before implementation.
- sdlc/    — IEC 62304 software development lifecycle phase documentation. Includes SOUP register template (`soup-register._TEMPLATE.md`).

## Key context
All software changes to SaMD must follow the SDLC process in sdlc/. Bug fixes that affect safety require a change control record in regulatory/change-control/.

## Not in this folder
- Regulatory submissions or design controls (→ regulatory/)
- Product requirements or PRDs (→ product/prds/)
- Clinical decisions (→ clinical/)
- Quality records — CAPA, complaints (→ quality/)
