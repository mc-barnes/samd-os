# Product Metrics

Metrics for tracking product value, adoption, and usage. These measure business outcomes, not regulatory compliance — see post-market/metrics.md for surveillance metrics.

## Key Metrics

| Metric | Definition | Target | Frequency |
|--------|-----------|--------|-----------|
| Active devices | Devices reporting data in the last 30 days | [target] | Weekly |
| User adoption | % of intended users actively using the device | > [threshold]% | Monthly |
| Feature utilization | % of key features used per session | > [threshold]% | Monthly |
| Mean time to result | Average time from input to actionable output | < [seconds] | Daily |
| User satisfaction (SUS) | System Usability Scale score | > 68 (above average) | Quarterly |
